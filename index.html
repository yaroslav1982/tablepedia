Article 2. Cryptocurrency exchange 2016 onward.

Cryptocurrency exchange (price in USD) since 2016:

http://tablepedia.com/cryptocurrency_exchange(01).html

Plots are generated with JavaScript commands as SVG file.

"2016-2020" button: Bitcoin, Etherium, XRP, BitcoinCash price history 2016 onward.

"Save SVG" button: download SVG file with plots.

"Save TABLE" button: download TXT file with cryptocurrency prices.

User can check/uncheck any of checkboxes to see plots, but only 4 of them are seen.

Article 1. Gold, silver, platinum and palladium price history 1900 onward.

Prices of Gold, Silver, Platinum, Palladium and Palladium (USD/OZ) since 1900:

http://tablepedia.com/metal_prices(01).html

Plots are generated with JavaScript commands as SVG file.

"1900-2020" button: gold, silver, platinum and palladium price history 1900 onward.

"1900-1970" button: gold, silver, platinum and palladium price history in 1900-1970.

"1970-2020" button: gold, silver, platinum and palladium price history 1970 onward.

"Save SVG" button: download SVG file with plots.

"Save TABLE" button: download TXT file with metal prices.

User can check/uncheck any of checkboxes to see plots, but only 4 of them are seen.
